# AI-Powered WordPress Blog Automation (n8n)

## Problem
Manually researching, writing, optimizing, and publishing SEO blog content is slow, expensive, and difficult to scale consistently.

## Solution
A fully automated n8n workflow that discovers topics, researches content, writes SEO-optimized articles using AI, generates media, and publishes directly to WordPress.

## What This System Does
- Discovers relevant topics automatically
- Scores relevance using AI
- Performs deep research
- Writes long-form SEO blog posts
- Generates or fetches images
- Publishes content directly to WordPress

## Workflow Breakdown
1. Topic discovery & vector storage  
2. AI-based relevance scoring  
3. Research & enrichment  
4. Article writing & SEO preparation  
5. Image generation & WordPress publishing  

## Tools Used
- n8n
- OpenAI (ChatGPT)
- HTTP & scraping tools
- Vector storage
- WordPress REST API

## Outcome
- Fully automated daily blog publishing
- Consistent SEO-structured content
- Zero manual writing or posting
- Scalable for agencies, SaaS, and content teams
